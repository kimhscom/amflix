## Features

- The list of movies and TV shows has been implemented as a carousel slider.
- Implement loading animation while load page
- Movies Features (Now playing, Upcoming, Popular)
- TV shows Features (Top Rated, Popular, Airing Today)
- Search for movies and TV shows
- It provides IMDB link, movie trailer, collections link, creators and movie stars information on the detail page
- Responsive web design
- Deployment to Netlify
